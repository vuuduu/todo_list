## Features of the To-Do List Application
Due Dates: 02/28/2024

### Task Group Management:
- **Add a new task group**: Create a new category to organize multiple to-do lists (e.g., "Party," "Work," etc.).
- **Update task group**: Modify the name or description of an existing task group.
- **Delete task group**: Remove a task group and all associated to-do lists.

### To-Do List Management:
- **Add a to-do list to a group**: Create a new to-do list within a task group.
- **Update a to-do list**: Modify the details of an existing to-do list (e.g., title, description).
- **Delete a to-do list**: Remove a to-do list from a task group.

### Task Management:
- **Add task to a to-do list**: Add individual tasks to a specific to-do list.
- **Update task**: Modify the details of an existing task (e.g., name, due date).
- **Delete task**: Remove a task from the to-do list.

### Additional Features:
- **Mark task as completed**: Allow users to mark tasks as completed, changing their status or appearance.
- **Sort and filter tasks**: Enable users to sort tasks by different criteria (e.g., priority, due date) and filter them based on specific attributes (e.g., completed, priority).
- **Set task priority**: Let users assign a priority level (e.g., high, medium, low) to tasks.
- **Task comments/notes**: Allow users to add extra details or notes to tasks for additional context or progress tracking.
- **Search for tasks**: Implement a search function that allows users to find tasks based on keywords or specific criteria.
